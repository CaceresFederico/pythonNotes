Conversion types

Implicit conversion

Sometimes python parser automatically converts some date to handle a few circumstances

num_str = "50"

# Manually converting string to integer

num_int = int(num_str)

print(num_int + 10)

# Output: 60

Explicit conversion

The programmer manually converts the data type using functions like int(), float(), str(), etc.

num_int = 10

num_float = num_int + 2.5

# Python converts `num_int` to float automatically

print(num_float)

# Output: 12.5

num_str = "50"

# Manually converting string to integer

num_int = int(num_str)

print(num_int + 10)

# Output: 60
